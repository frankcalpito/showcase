# Showcase

This shows some custom components built using next, typescript, tailwind, and storybook. 

### Run Storybook

```bash
npm run storybook
# or
yarn storybook
```

### Build Static Storybook

```bash
npm run build-storybook
# or
yarn build-storybook
```

### Cypress testing

```bash
npm run cypress
# or for cli only 
npm run test
npm run test:e2e
```
